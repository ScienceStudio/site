List of technologies I use

NVIDIA CUDA link

This is one of the cheapest and powerful computation platforms for making programs run significantly faster. The main idea is that grphics processor (GPU) has more of computational kernels than computer CPU.

Pros: Depending on algorithms, used GPUs, cooling sytem (no joke!) speedup of computations can vary from 10x to 1000x.

Cons: High correlation of computation speed and price of GPU, not so easy to make programs, GPUs can burn out (on macbook - definitely will), runs only NVIDIA cards, many troubles with drivers on *nix, hard to test.

Resume: Despite all cons, I like this thing: you can increase speed of application running using only refactoring (neither very difficult nor easy to make btw) and putting nice gaming videocard in computational system. If you like burning machines for grinding huge amounts of data, this is for you.

Java OpenGL (JOGL) link

Everybody know that 3D graphics is connected mostly to 2 graphical API platforms: Direct3d or OpenGL. OpenGL is free and can be used everywhere, so I prefer it. OpenGL is used mostly in pair with C++ or C, but there are a lot of business or scientific applications written in java, and sometimes there is trouble in real time visualization of large amounts of data, so using of OpenGL is the most easy and obvious solution. For using OpenGL with java I prefer JOGL binding. JOGL is cool.

Pros: OpenGL in java apps!

Cons: Quite buggy, dependable on hardware, drivers and operating system and requires understanding of OpenGL to make app crossplatform and help app to survive in uncontrollable environment.

Resume: Very nice solution for making 3d apps.

WebGL using three.js link

There is a demand for high perfomance ultra nice looking web sites. One of solutions is to use canvas, but sometimes pure canvas works slowly and can't give enough functionality - that is a ring to use WebGL. But there are a huge amount of functions - and that is a ring to use framework. One I prefer is three.js because of speed and easiness to understand: you can use it even without deep understanding of OpenGL. Fast and reliable,

Pros: 3d app in web! Easy, fast to develop.

Cons: Quite buggy, dependable on build of browser, hardware, drivers and operating system. Eats a lot of RAM and heats CPU and GPU.

Resume: Web sites looks very beautiful, technology is fast growing. Go and use it now.
